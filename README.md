
## Getting started in 10 minutes

- Clone this repo 
- Install requirements
- Run the script
- Browser http://127.0.0.1:5000
- Done!

### Clone the repo
```shell
$ git clone git@github.com:tranhoangcore/age-gender-detection.git
```

### Install requirements

```shell
$ pip install -r requirements.txt
```
### Run with Python

```shell
$ python app.py
```
